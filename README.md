# TBNP Custom Web

Custom frontend for Bungoma National Polytechnic. Built with Next.js and Tailwind CSS.

## Portals

- **Main Website** - bungomapoly.ac.ke
- **Student Portal** - students.bungomapoly.ac.ke
- **Staff Portal** - staff.bungomapoly.ac.ke
- **Admin Portal** - admin.bungomapoly.ac.ke

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed to Vercel automatically from `main` branch.
